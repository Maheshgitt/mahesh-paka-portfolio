Problem Statement

Why this project?

Understanding analog modulation techniques like AM and FM is fundamental to communications engineering, but most learning is limited to theory or expensive lab equipment such as oscilloscopes and signal generators that aren't always accessible.

There was a need for an affordable, portable, embedded platform that could capture real audio, apply modulation/demodulation and digital signal processing algorithms, and display the resulting waveforms visually in real time — making signal analysis hands-on and accessible.

The Project

What I built

An ESP32-based audio capture and modulation analysis system with live waveform visualization.

Built an ESP32-based system to capture audio via the INMP441 I2S microphone and visualize real-time waveforms on a 2.8″ SPI TFT display. The system implements AM/FM modulation analysis with optimized embedded DSP routines for educational and practical applications.

Key Features

  • Real-time audio capture using I2S protocol with INMP441 microphone
  • Live waveform visualization on 2.8″ SPI TFT display
  • AM/FM modulation and demodulation algorithms
  • Down sampling with optimized DSP routines
  • User-friendly interface for signal analysis

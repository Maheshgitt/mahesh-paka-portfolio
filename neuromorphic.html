Problem Statement

Why this project?

Traditional von Neumann computing architectures separate memory and processing, leading to high power consumption and bottlenecks when running brain-inspired, event-driven workloads such as spiking neural networks. As AI workloads grow, there is increasing demand for hardware that can natively emulate neural behavior at the transistor level.

This project set out to design and simulate analog circuits — at the synapse and neuron level — that mimic biological spike generation and signal transmission, forming a foundational building block for low-power, brain-inspired (neuromorphic) computing systems.

The Project

What I built

A transistor-level neuromorphic synapse and neuron cell built and simulated in 90nm CMOS.

Designed a transistor-level neuromorphic synapse and neuron cell to emulate biological spike behavior. This project explores brain-inspired computing architectures that can process information more efficiently than traditional von Neumann systems, implementing a current-mirror output stage to convert synapse activity into measurable neuron spike responses.

Key Features

  • Implemented using 90nm CMOS technology for optimal performance
  • Current-mirror output stage for measurable neuron spike responses
  • Biological spike behavior emulation with tunable parameters
  • Low power consumption design suitable for edge computing
  • Scalable architecture for neural network implementations

Technologies Used

What's Next

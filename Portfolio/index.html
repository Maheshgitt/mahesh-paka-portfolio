Neuromorphic VLSI

Neuromorphic Neuron–Synapse Circuit

December 2025

Project Overview

Designed a transistor-level neuromorphic synapse and neuron cell to emulate biological spike behavior. This project explores brain-inspired computing architectures that can process information more efficiently than traditional von Neumann systems.

Key Features

  • Implemented using 90nm CMOS technology for optimal performance
  • Current-mirror output stage for measurable neuron spike responses
  • Biological spike behavior emulation with tunable parameters
  • Low power consumption design suitable for edge computing
  • Scalable architecture for neural network implementations

Technologies Used

Cadence Virtuoso Spectre Simulator 90nm CMOS Analog Design SPICE
Embedded DSP

SignalHub – Audio Modulation Analyzer

November 2025

Project Overview

Built an ESP32-based system to capture audio via INMP441 (I2S) and visualize real-time waveforms on a 2.8″ SPI TFT display. The system implements AM/FM modulation analysis with optimized embedded DSP routines for educational and practical applications.

Key Features

  • Real-time audio capture using I2S protocol with INMP441 microphone
  • Live waveform visualization on 2.8″ SPI TFT display
  • AM/FM modulation and demodulation algorithms
  • Down sampling with optimized DSP routines
  • User-friendly interface for signal analysis

Technologies Used

ESP32 INMP441 I2S Protocol DSP AM/FM Modulation SPI TFT

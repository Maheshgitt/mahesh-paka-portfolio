Problem Statement

Why this project?

Tech enthusiasts and professionals struggle to keep up with the constant flow of technology news scattered across dozens of websites and apps. Reading full-length articles is time-consuming, and most existing news apps offer no way to interact with the content, ask follow-up questions, or get a quick digest of what an article actually says.

There was a clear need for a single, lightweight Android application that aggregates live tech headlines, condenses them into quick AI-generated summaries, notifies users of important updates in real time, and lets users converse naturally with an AI assistant about the news they're reading.

The Project

What I built

A feature-complete AI-powered Android news application with a Python FastAPI backend deployed on Render.

The app fetches live tech headlines via the NewsData.io API, generates AI summaries using the Groq LLM API, sends push notifications through Firebase Cloud Messaging, and provides an in-app AI chatbot with persistent conversation history — all delivered through a modern Jetpack Compose UI backed by a Retrofit-driven FastAPI service.

Key Features

  • Live tech news feed powered by the NewsData.io API
  • Groq LLM integration for one-tap AI article summaries
  • Firebase Cloud Messaging for background push notifications
  • In-app AI chatbot with persistent chat history
  • FastAPI backend deployed on Render with RESTful endpoints
  • Modern Jetpack Compose UI with Material 3 design

Technologies Used

Kotlin Jetpack Compose FastAPI Groq LLM Firebase FCM NewsData.io Render Retrofit

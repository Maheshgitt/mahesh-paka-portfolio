BER Checker – FPGA Hamming(7,4) SECDED

February 2026  •  Verilog, Xilinx Vivado, Nexys 4 Artix-7

A complete Bit Error Rate checker implemented on the Nexys 4 Artix-7 FPGA board using Xilinx Vivado. The design evolved from basic XOR error detection through to a full Hamming(7,4) SECDED (Single Error Correction, Double Error Detection) implementation, with four selectable display modes, BTNC-triggered reset, and a comprehensive testbench with all 33 test cases verified and passing.

Problem Statement

Why this project?

Digital communication channels are inherently susceptible to noise, which can corrupt transmitted bits. Simple parity-based error detection can flag that an error occurred but cannot correct it, forcing costly retransmissions and reducing effective throughput.

There was a need for a hardware-level error control scheme that could not only detect but also correct single-bit errors, detect double-bit errors, and provide a way to visualize and measure the resulting bit error rate directly on FPGA hardware — verified rigorously against a full testbench.

The Project

What I built

A Hamming(7,4) SECDED based BER checker implemented and verified on the Nexys 4 Artix-7 FPGA.

The design evolved from a basic XOR-based error detector into a full Hamming(7,4) Single Error Correction, Double Error Detection (SECDED) implementation. It includes four selectable display modes shown on the board's 7-segment displays, a debounced BTNC hardware reset, and a complete testbench in which all 33 test cases were verified and passed.

Key Features

  • Hamming(7,4) SECDED for single-bit correction and double-bit detection
  • Four display modes for real-time BER monitoring on 7-segment display
  • BTNC hardware reset with debounce logic
  • All 33 testbench cases verified and passing
  • Synthesized and implemented on Nexys 4 Artix-7 FPGA
  • Modular Verilog design with clean timing constraints

Technologies Used

Verilog Xilinx Vivado Nexys 4 Artix-7 Hamming(7,4) SECDED FPGA Testbench
